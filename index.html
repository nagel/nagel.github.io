Budget Buddy

Budget Buddy is a web application designed to manage your personal monthly budget. Utilizing the Plaid API, users have the ability to pull information from financial accounts linked to the application. After accounts have been linked, budget buddy allows users to create custom categories for their transactions. Feel free to clone the repositories below and demo the application!

Front-End Repository
Back-End Repository
Dashboard

Users have the ability to view categorized transactions, create new categories, re-categorize transactions, and create a budget amount for each category. Thanks to Vue.js, info is dynamically updated on the page after changes are made.

Transactions

Users can view a master list of transactions that are updated upon login. Similar transactions that have been previously assigned a category will automatically be categorized.


Accounts

Accounts are organized by subtype which includes checking, savings, credit card, money market, and more. The page populates after a user links their account(s) to the application and updates amounts via the Plaid API.
